1 Executive Summary

On easy-to-moderate work, all three are excellent and Gemma-4-31B wins on value — perfect Tier-1 score, ~2× faster, and free. However, on hard, multi-layered work, reliability separates the frontier models from the rest.

gpt-oss-120B is the reliability champion: it achieved a perfect score on every Tier-2 task across all three trials, including the full-stack app (3/3). Gemma-4-31B is fast but inconsistent on the most complex tasks — its server failed to start in 2 of 3 trials due to hardcoding the port instead of honoring PORT.

Default: Gemma-4-31B → Escalate: gpt-oss-120B

2 Methodology

The benchmark is execution-graded with partial credit. Each project is scored by running the result against hidden checks. For the full-stack app, the server is actually spawned and hit over real HTTP (GET/POST/PATCH/DELETE, status codes, content types, persistence).

Tier 1 (N=1): Everyday tasks (KV store, Debugging, Async, CLI). Tier 2 (N=3): Advanced tasks (Web Framework, State Store, Full-stack App). Tier 2 used three trials to capture run-to-run variance.

5 Deep-Dive — the Full-Stack Application

The centerpiece task asks for a complete web app in Node built-ins only: a REST API, a tasks.json persistence layer, and an HTML+JS frontend. The grader spawns the server on a fresh port and drives all ten behaviors over real HTTP.

gpt-oss-120B and MiniMax-M2.7 delivered correct, fully working apps in every trial. Gemma-4-31B produced correct-looking code, but the server only ran when the port matched its hardcoded default. This operational gap (ignoring the injected port) is where fast, cheap models typically slip.

6 Limitations & Caveats

N=3 for Tier 2 Directional reliability, not precise probabilities.
Bounded Projects Small self-contained tasks, not thousand-line codebases.

8 Appendix — Compensation & Mitigation

Gemma's failures were caused by nested backticks in large HTML strings. By introducing targeted compensation (serving HTML from separate files, improved repair diagnostics, and a budget increase from 3 to 5 repair rounds), Gemma was brought to a consistent 100%.

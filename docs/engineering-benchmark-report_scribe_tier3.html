Deep Dive: Full-Stack Failure & Fix

Gemma-4-31B initially failed the full-stack task in 2/3 trials due to hardcoded ports and template-literal syntax errors.

The Diagnosis

Failures were caused by nested backticks in large inline HTML strings, creating Syntax Error crashes before the server could bind to process.env.PORT.

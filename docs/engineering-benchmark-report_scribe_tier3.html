Deep Dive: The Full-Stack Gap

The Full-Stack application is the critical differentiator. It requires a REST API, JSON persistence, and a served frontend, all coordinated via an operational contract (e.g., honoring process.env.PORT).

Success Mode

gpt-oss & MiniMax: Delivered fully working apps in 100% of trials. Routing, REST semantics, and integration were flawless.

Failure Mode

Gemma: Logic was often correct, but failed the operational contract by hardcoding port 3000, causing 2/3 trials to fail to start.
